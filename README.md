# Finance App for Desktops !!!

A finance management app for Windows, Linux and MacOS that uses Plaid services to collect user bank data, like transactions and payments, and creates user-friendly and informative UI.

# How to run

1. npm run npm-install-packages
2. npm run start
3. For individual processes -
   "fe": "cd frontend && npm run dev",
   "be": "cd backend && npm run dev",
   "electron": "cd electron && npm run dev",
   "npm-install-packages": "cd frontend && npm i && cd ../backend && npm i && cd ../electron && npm i"
